🖼️ MAQUETTES VISUELLES – AYAN BRIDGE VERSION 2 
📌 Objectif : Présenter une vision claire des interfaces utilisateurs principales pour la plateforme Ayan Bridge Version 2, selon les pôles et modules définis. 

🏠 PAGE D'ACCUEIL (HOMEPAGE) 
🎯 Objectif : Présenter la promesse de la plateforme et guider l'utilisateur selon son profil ou son intention. 

Bandeau principal avec slogan : "Apprends. Crée. Investis. Gagne." 
5 boutons clairs pour accéder selon l’intention de l’utilisateur : 
   📚 Je veux explorer la bibliothèque 
   🧠 Je veux investir 
   🎓 Ayan Learn Hub 
   🎛️ Salle d’opérations 
   📺 Bridge Live 
Scroll-down avec 3 sections explicatives : 
   - Comment ça marche (boucle d'évolution) 
   - Produits populaires 
   - Témoignages clients 
🔵 MAGASIN (MARKETPLACE) 
🎯 Objectif : Explorer les produits disponibles à l’achat ou en abonnement. 

Barre de recherche avancée (titre, thème, type, niveau, prix) 
Filtres : catégorie, format (ebook, vidéo, live…), créateur 
Affichage : vignettes de produits avec : 
   - Image de couverture 
   - Titre 
   - Prix 
   - Bouton "Voir +" ou "Acheter" 
   - 🔴 Indicateur LIVE si une session est en direct liée au produit 
🟠 STUDIO INTELLIGENT 
🎯 Objectif : Permettre à un utilisateur de créer un produit numérique facilement. 

Outils intégrés pour la création : 
   - 🧑‍🏫 Whiteboard collaboratif 
   - 🎤 Enregistreur audio/vidéo intégré 
   - 📚 Éditeur de cours avec structuration en modules, leçons, quiz 
   - 🎞 Scénarisation de formation (diapositives + narration) 
   - 📥 Upload de ressources (PDF, images, vidéos) 
   - 🎬 Mode "Enregistrer & Publier" ou "Diffuser en direct" 
Étapes visuelles (type tunnel) : 
   1. Choisir type de produit (eBook, fiche, mini-formation, live…) 
   2. Saisir l'idée ou thème 
   3. Génération automatique (IA intégrée) 
   4. Modifier / valider / publier 
   5. Mise en vente directe ou en session live 
UI : Interface glisser-déposer avec sidebar + preview live du rendu PDF ou préparation de session LIVE 
🟢 Ayan Learn Hub 
🎯 Objectif : Espace d’apprentissage centralisé pour suivre des parcours éducatifs, des cours scolaires, des formations spécialisées ou des sessions en direct. 

Composants généraux : 
   - Tableau de bord personnel avec progression 
   - Menu par niveaux (primaire, secondaire, pro…) 
   - Recommandations de parcours, suivi IA, rappels 
Outils live intégrés pour l’enseignement : 
   - 🧑‍🏫 Whiteboard en direct (intégré à chaque session) 
   - 💬 Outils interactifs : quiz en direct, annotations, partages d’écran 
   - 🗓️ Planification des cours (live ou enregistrés) 
   - 🎥 Génération de replays et certificats automatiques 
🔹 École virtuelle (Bridge School) : 
   - Cours organisés par niveau, semaine, matière 
   - Exercices, tests, progression par badges 
   - Sessions LIVE par classe ou enseignant en temps réel (visioconférence intégrée) 
   - Boutons : "Commencer", "Reprendre", "Rejoindre en direct" 
🔹 Académie (Bridge Academy) : 
   - Formations thématiques en modules (développement personnel, business, parentalité...) 
   - Certifications à la fin de parcours 
   - Sessions LIVE interactives avec formateurs / tuteurs 
🔹 Sessions en direct externes (Bridge Live) : 
   - Liste d’événements produits par Ayan Bridge : Webinaires, ateliers, conférences, podcasts, émissions thématiques 
   - Fiches événement : titre, date, intervenant, résumé, bouton "S’inscrire" 
   - Intégration directe (Zoom, Jitsi, ou plateforme interne) 
   - Accès aux replays classés par thème ou série 
   - Option "Diffusion continue" façon chaîne TV éducative Ayan Bridge 
💼 BRIDGE CAPITAL / INVESTOR CLUB 
🎯 Objectif : Proposer des opportunités d’investissement numérique 

UI : 
   - Tableau des produits "sponsorisables" 
   - Fiche produit d’investissement : 
       - Titre / description / gains potentiels / créateur associé 
       - % de retour estimé / historique des ventes 
       - Bouton "Investir maintenant" 
       - Indicateur de lancement LIVE si projet présenté en temps réel 
🧠 PROFIL UTILISATEUR UNIQUE 
🎯 Objectif : Unifier l’expérience autour d’un tableau de bord unique 

Sections : 
   - Mon profil (infos, rôle, préférences) 
   - Mes achats 
   - Mes créations 
   - Mes ventes / revenus affiliés 
   - Mes investissements 
   - Notifications 
   - Mes sessions LIVE (planifiées / en cours / replays)
